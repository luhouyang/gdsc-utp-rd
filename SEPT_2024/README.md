There is an integer $i$ that iterates from $i = 0$ &rarr; $i = 100$

Write a program that outputs "$i$ foo" when $i$ is **even** and "$i$ bar" when $i$ is **odd** on a new line. Replace $i$ with the current value of $i$.

<br />

<div>
    <p align="center">
        <b>Expected Output</b>
    </p>
</div>

---

<br />

```
0 foo
1 bar
2 foo
3 bar
.
.
.
```
